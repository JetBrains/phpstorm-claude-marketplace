# Contributing

Thanks for your interest in contributing!

This repository contains Markdown skills and agents, Bash hooks, and JSON configuration. We welcome:

- Fixes to typos and grammar
- Improvements to clarity and structure
- New skills, agents, hooks, or examples that fit the existing scope

Before you start, please:

1. Make sure your changes keep the tone and style of the existing text.
2. Follow existing patterns for hooks and skills.
3. Avoid adding internal or confidential information.

## How to contribute

1. Fork the repository.
2. Create a branch for your change.
3. Make your edits.
4. Test hooks and skills with a real PhpStorm + Claude Code setup if possible.
5. Open a pull request.

In your pull request description, briefly explain:

- What you changed
- Why it improves the project
